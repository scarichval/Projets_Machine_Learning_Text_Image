Projet d'Analyse de Classification

Ce projet a pour but de comparer la performance de plusieurs algorithmes de classification sur un jeu de données Twitter. Le jeu de données comprend des utilisateurs légitimes et des pollueurs de contenu. Les étapes suivantes guident à travers le processus de prétraitement des données, la sélection des caractéristiques, et la comparaison des algorithmes de classification.

Étapes pour exécuter le projet

Exécuter : (NB: Il y a un petit délai d'environs une minute lors de l'execution de la commande, veuillez patienter)
python3 main.py

Au cours de l'exécution du programme, des fenêtres pop-up successives apparaîtront, chacune faisant partie intégrante du processus.
L'exécution sera suspendue en attendant la fermeture de chaque pop-up. Lorsqu'une fenêtre est fermée, la suivante s'affichera jusqu'à la fin du processus. 

Ensuite, les étapes suivantes s'executeront a tour de role.


1. Prétraitement des données

Avant de commencer les analyses, il est nécessaire de prétraiter les données. Le script suivant effectuera le nettoyage, la normalisation, et la préparation des ensembles de données pour les étapes suivantes.

2. Comparaison des algorithmes de classification

Une fois que les données sont prétraitées, le script suivant compare différents algorithmes de classification, tels que l'arbre de décision, la forêt aléatoire et la classification bayésienne naïve.

3. Sélection des caractéristiques

Pour optimiser le modèle de classification, on peut sélectionner les caractéristiques les plus pertinentes. Ce script utilise le gain d'information pour identifier les 7 caractéristiques les plus importantes.


En suivant ces instructions, vous pourrez exécuter les différents scripts du projet et obtenir les résultats des analyses de classification.
